## HOW TO RUN SERVER

1. ketik `cd server` untuk masuk ke dir server
2. ketik `npm install`
3. setting database dulu dengan melihat petunjuk di `/server/.env`
4. ketik `npm start` pada terminal
5. happy collaboration
